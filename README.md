# NTI-Freelance-Tasks

This repository contains a collection of micro-projects developed during the NTI Freelance training. Each project is organized in its own directory.

---

##  Table of Contents
1. [User Profile Viewer (Codezilla)](#1-user-profile-viewer)

3. [Product Listing (Hariga)](#3-product-listing)

---

## 1. User Profile Viewer
**Client:** Codezilla SRL
**Tech Stack:** Angular 21, Node.js, Express, MongoDB.
- **Description:** A simple app to fetch and display hardcoded user profiles from a database.
- **Folder:** `/user-profile-viewer`

## 2. Static Content Display
**Client:** Tuio
**Tech Stack:** Angular 21, Express.js.
- **Description:** Serving static HTML/CSS content using an Express server without a database.
- **Folder:** `/static-content-app`

## 3. Product Listing
**Client:** Hariga
**Tech Stack:** MEAN Stack.
- **Description:** Displaying a dynamic list of products with a clean UI.
- **Folder:** `/product-listing-app`
